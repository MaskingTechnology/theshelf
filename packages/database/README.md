
# Database | The Shelf

The database packages provide a universal driver system for working with data storage solutions.

It's based on simple CRUD operations and purposely does NOT support relational querying.

## Packages

* **[Core](./core/README.md)** - definition of the CRUD operations and in-memory driver
* **[MongoDB driver](./drivers/mongodb/README.md)** - driver for [MongoDB](https://mongodb.com)
